# [JavaScript] timezone 확인하는 방법

## timezone 확인하는 방법

클라이언트의 timezone을 서버에 넘겨주어야 하는 경우가 있습니다.  
자바스크립트로 클라이언트의 timezone을 구하는 방법은 간단합니다.

### timezone 확인 방법

```js
const timezone = Intl.DateTimeFormat().resolvedOptions().timeZone;
console.log("timezone : ", timezone);
```
