# [JavaScript] FormData Json 데이터로 변환하기, Json 데이터 FormData 변환하기

http 요청을 하다 보면 body 데이터에 json 데이터, form data를 넣는 경우가 있다.  
**form data에서 json, json에서 form data**로 자료를 변환할 일이 있어서 간단하게 정리해보았다.

## FormData를 Json 객체로 변환하기

```js
const formData = new FormData();
formData.append("apple", "사과");
formData.append("banana", "바나나");
formData.append("cherry", "체리");

const json = Object.fromEntries(Array.from(formData.entries()));

console.log(json);
```

## Json 객체를 FormData로 변환하기

```js
const json = {
  apple: "사과",
  banana: "바나나",
  cherry: "체리"
};

const formData = new FormData();

Object.keys(json).forEach(key => {
  formData.append(key, json[key]);
});

for (const value of formData.entries()) {
  console.log(value[0] + ", " + value[1]);
}
```
