# [Visual Studio] 설치된 Visual Studio 버전 확인하기

## 버전 확인 방법

설치된 Visual Studio 버전을 확인하는 방법은 어렵지 않다. **Visual Studio를 실행**하고 **도움말(H) 클릭** -> **Microsoft Visual Studio 정보 클릭** 하면 끝이다.

/>

클릭하면 아래와 같이 Microsoft Visual Studio 정보 팝업이 표시된다. **Visual Studio, .Net Framework 버전, 설치된 제품 등** 여러 정보가 표시된다.
