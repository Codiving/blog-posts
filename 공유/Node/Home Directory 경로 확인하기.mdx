# [Node] Home Directory 경로 확인하기

nodejs 개발하면서 **home directory**가 필요한 경우가 있습니다.  
그러나 OS별로 home directory 경로가 다르기 때문에 직접 처리하려면 일일이 작성해주어야 합니다.  
Home directory 경로는 **node에서 제공해주는 함수**를 이용하면 쉽게 얻어올 수 있습니다.

## Home directory 경로 확인

아래 코드를 실행하면 끝입니다.

```js
const homedir = require("os").homedir();
console.log("homedir : ", homedir);
```
