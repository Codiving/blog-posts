# [Node] 경로 구분자 확인하기

nodejs 개발하면서 **파일 경로 구분자**를 확인 해야하는 경우가 있습니다.  
그러나 OS 별 파일 경로 구분자가 다르기 때문에 직접 처리하려면 일일이 작성해주어야 합니다.  
파일 경로 구분자는 **node에서 제공하는 함수**를 이용하면 쉽게 얻어올 수 있습니다.

## OS별 파일 경로 구분자 확인하기

아래 코드를 실행하면 끝입니다.

```js
const path = require("path");
console.log("path : ", path.sep);
```
